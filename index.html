Meus conhecimentos

PHP

É uma linguagem de programação de código aberto amplamente utilizada para desenvolvimento web. Ele é executado no lado do servidor e é capaz de gerar conteúdo dinâmico para páginas da web. O PHP é fácil de aprender e usar, e é compatível com uma ampla variedade de bancos de dados, incluindo MySQL e PostgreSQL

JavaScript

Linguagem de programação amplamente utilizada para tornar as páginas da web interativas. Ela permite a criação de scripts que são executados no navegador do usuário sendo considerada uma linguagem de script de alto nível.

SQL

SQL é usado para gerenciar e manipular bancos de dados relacionais. É a linguagem padrão para interagir com sistemas de gerenciamento de bancos de dados (DBMS) como MySQL, PostgreSQL, SQL Server, entre outros. Com SQL, podemos realizar operações como inserir, atualizar, excluir e consultar dados em bancos de dados, tornando-o essencial para o desenvolvimento de aplicativos que envolvam armazenamento e recuperação de informações.

HTML 5

HTML é a linguagem de marcação usada para criar a estrutura básica de uma página da web. É composta por elementos que são interpretados pelo navegador para exibir conteúdo na página. Juntamente com o CSS e o JavaScript, o HTML forma a base do desenvolvimento web e é essencial para criar páginas da web estáticas e dinâmicas.

CSS 3

CSS é uma linguagem de estilo utilizada para definir a aparência e o layout dos elementos HTML em uma página da web. O CSS é aplicado de forma cascata, o que significa que as regras podem ser sobrepostas e herdadas, permitindo um controle preciso sobre o design de uma página da web.

Bootstrap

Bootstrap é um framework de código aberto para desenvolvimento front-end de sites e aplicativos da web. Ele fornece um conjunto de estilos predefinidos, componentes e ferramentas de layout responsivo que facilitam a criação de interfaces de usuário atraentes e compatíveis com dispositivos móveis.
